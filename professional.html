Robotics Projects

Multi-Core Motor Encoder Driver — Autonomous Robotics

Developed an ESP32-based motor encoder driver using FreeRTOS, configuring PCNT hardware for quadrature encoding and transmitting timestamped data over UART to a Jetson processor. Architected a dual-core pipeline with pinned tasks for 50 Hz encoder sampling, using inter-task queues to isolate time-critical motor feedback from communication overhead.

Kalman Filter IMU Integration — Mars Rover

Cal Poly's Mars Rover relies solely on wheel encoders for motion detection — a problem when caught in a sandtrap, where encoders report movement even when the rover is stuck. I implemented an Error-State Kalman Filter (ESKF) fusing accelerometer and gyroscope data for real-time attitude estimation, and am developing wheel-slip fault detection by comparing IMU motion estimates against encoder feedback to identify sand entrapment.
